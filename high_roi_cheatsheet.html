🗣️ TOP 15 HIGH-ROI SENTENCE PATTERNS

Pattern 1: Heb je...? (16×) - HIGHEST ROI!

Use: Ask if someone has/wants/feels like something

Heb je tijd? = Do you have time?
Heb je zin om mee te gaan? = Do you feel like coming along?
Heb je plannen? = Do you have plans?
Heb je honger? = Are you hungry?

Grammar: heb + je + noun/om te + infinitive

Pattern 2: Zullen we...? (5×) - Make Suggestions

Use: Suggest doing something together

Zullen we afspreken? = Shall we meet?
Zullen we pasta maken? = Shall we make pasta?
Zullen we naar het park gaan? = Shall we go to the park?

Grammar: Zullen + we + infinitive

Pattern 3: Ik ga... (12×) - Talk About Plans

Use: Say what you're going to do

Ik ga naar huis = I'm going home
Ik ga werken = I'm going to work
Ik ga met de trein = I'm going by train

Grammar: Ik ga + place/infinitive

Pattern 4: Wat vind je van...? (7×) - Ask Opinions

Use: Get someone's opinion

Wat vind je van het weer? = What do you think of the weather?
Wat vind je van deze film? = What do you think of this movie?

Grammar: Wat vind je van + noun

Pattern 5: Hoe...? (16×) - Ask "How" Questions

Use: Ask about manner/degree

Hoe laat? = What time?
Hoe vaak? = How often?
Hoe lang? = How long?
Hoe kom je naar...? = How do you get to...?

Pattern 6: Ik hou van... (8×) - Express Likes

Use: Say what you like

Ik hou van honden = I like dogs
Ik hou van sport = I like sports
Ik hou niet van kou = I don't like cold

Grammar: Ik hou (niet) van + noun

Pattern 7: Waar...? (10×) - Ask "Where"

Use: Ask about location

Waar woon je? = Where do you live?
Waar werk je? = Where do you work?
Waar is...? = Where is...?

Pattern 8: Dat is... (12×) - Describe/Explain

Use: State facts or descriptions

Dat is leuk! = That's nice!
Dat is waar = That's true
Dat is fijn = That's good/nice
Dat is snel = That's fast

Pattern 9: Ik vind... (11×) - Give Opinions

Use: Express your opinion

Ik vind het leuk = I think it's nice
Ik vind het moeilijk = I find it difficult
Ik vind hem saai = I find him boring

Pattern 10: ... per week - Frequency Expressions

Use: Talk about frequency

drie keer per week = three times per week
twee dagen per week = two days per week
elke week = every week

Pattern 11: Mag ik...? - Ask Permission

Use: Politely request

Mag ik hem passen? = May I try it on?
Mag ik vragen? = May I ask?

Pattern 12: ... graag - Politeness Marker

Use: Make requests polite

Ik zou graag... = I would like...
Graag een kopje thee = A cup of tea, please

Pattern 13: Hoelang/Hoeveel...? - Quantity/Duration

Use: Ask about amounts/time

Hoelang duurt het? = How long does it take?
Hoeveel kost dat? = How much does that cost?
Hoeveel dagen? = How many days?

Pattern 14: Ik moet... (7×) - Express Obligation

Use: Say what you have to do

Ik moet werken = I have to work
Ik moet gaan = I have to go

Pattern 15: ... of niet? - Confirmation Tag

Use: Confirm understanding

Dat is goed, of niet? = That's good, isn't it?
Je komt, of niet? = You're coming, aren't you?

📝 GRAMMAR ESSENTIALS

Focus: ~25% - Learn patterns, not rules!

1. Present Tense (Most Important!)

Regular Verbs: stem + t (jij/hij), stem + en (wij/jullie/zij)
werken (to work): ik werk, je werkt, hij werkt, we werken
Top Irregular Verbs (MEMORIZE!):
zijn: ben, bent, is, zijn
hebben: heb, hebt, heeft, hebben
gaan: ga, gaat, gaan
kunnen: kan, kunnen
willen: wil, willen
moeten: moet, moeten

2. Word Order Basics

Main clause: Subject + Verb + Rest
Ik werk in Amsterdam = I work in Amsterdam
Question: Verb + Subject + Rest
Werk je in Amsterdam? = Do you work in Amsterdam?
With omdat/als/dat: Subject + Rest + Verb (LAST!)
...omdat ik in Amsterdam werk = ...because I work in Amsterdam

3. Past Tense (Perfectum) - Common Pattern

Pattern: hebben/zijn + ge...t/d or ge...en
Ik heb gewerkt = I (have) worked
Ik ben gegaan = I (have) gone
Top 10 Irregular Past Participles:
gedaan (done), geweest (been), gehad (had), gegaan (gone),
gekomen (come), gezien (seen), gegeten (eaten), gedronken (drunk),
gekregen (gotten), geworden (become)

4. Modal Verbs (Can/Want/Must)

Pattern: Modal + infinitive (end of sentence)
Ik kan zwemmen = I can swim
Ik wil gaan = I want to go
Ik moet werken = I must work

5. Om te... (in order to)

Pattern: om + te + infinitive
Ik kom om te leren = I come (in order) to learn
Tijd om te eten = Time to eat

🎓 EXAM-SPECIFIC TIPS

1. Vocabulary Test (20 MC, 15 min)

  • Focus: additionalvocab.md words (high priority!)
  • Strategy: Learn most frequent nouns, verbs, adjectives first
  • Passing: 12/20 points minimum

2. Listening Test (20 MC, 20 min)

  • Key skill: Recognize question patterns above
  • Practice: Listen for Heb je...?, Wat...?, Waar...?, Hoe...?
  • Passing: 14/20 points minimum

3. Reading Test (20 MC, 45 min)

  • Focus: Main ideas, don't get stuck on unknown words
  • Key: Connecting words (maar, omdat, dus) show logic
  • Passing: 14/20 points minimum

4. Writing Test (5 sentences + short text, 30 min)

  • Use: Sentence patterns from this guide!
  • Grammar counts: 1 point for content, 1 for grammar
  • Safe patterns: Ik..., Ik ga..., Ik vind..., Omdat..., Maar...
  • Passing: 15/22 points minimum

💡 PRO TIPS

Use "Ik ga..." for everything: "Ik ga werken", "Ik ga eten", "Ik ga naar huis"
When in doubt, use "Heb je...?": Works for questions about almost anything
"Dat is..." is your friend: Easy way to comment on anything
Add "graag" to be polite: "Ja graag", "Ik zou graag..."
Combine patterns: "Heb je zin om mee te gaan omdat het mooi weer is?"

Success Formula
