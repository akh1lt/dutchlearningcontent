### B. Essential Sentence Patterns (From Real Dialogues)

#### **PATTERN 1: Making Requests/Asking Permission**
```
Kun je [activity]?          - Can you [do something]?
Mag ik [activity]?          - May I [do something]?
Wil je [activity]?          - Will you/Do you want to [do something]?
Kun je me helpen met...?    - Can you help me with...?
```

**Real Examples from Texts:**
- Kun je Joris naar het hockeyveld brengen? (Can you take Joris to the hockey field?)
- Mag ik je iets vragen? (May I ask you something?)
- Wil jij die vuile borden afwassen? (Will you wash those dirty dishes?)
- Kun je even met me meedenken? (Can you think along with me?)

**Practice:** Memorize these as complete chunks. Don't think "kun" + "je" + verb. Think: "Kun je..." = request pattern.

---

#### **PATTERN 2: Expressing Needs/Wants**
```
Ik moet...                  - I have to/must...
Ik wil (graag)...          - I want (to)...
Ik zou graag...            - I would like to...
Ik heb ... nodig           - I need...
```

**Real Examples:**
- Ik moet Joris naar hockey brengen (I have to take Joris to hockey)
- Ik wil geen kat (I don't want a cat)
- Ik zou graag Nederlands leren (I would like to learn Dutch)
- Ik heb je advies nodig (I need your advice)

---

#### **PATTERN 3: Giving Opinions/Feelings**
```
Ik vind (dat)...           - I think/find (that)...
Dat lijkt me...            - That seems to me...
Ik hou (niet) van...       - I (don't) like...
Ik ben dol op...           - I love...
Het is...                   - It is...
```

**Real Examples:**
- Dat lijkt me heerlijk (That seems lovely to me)
- Ik vind lezen geweldig (I think reading is great)
- Ik hou niet van honden (I don't like dogs)
- Het is lastig (It's difficult)

---

#### **PATTERN 4: Questions (Most Common Starters)**
```
Hoe...?                     - How...?
Wat...?                     - What...?
Waar...?                    - Where...?
Wanneer...?                 - When...?
Waarom...?                  - Why...?
Heb je...?                  - Do you have...?
Ben je...?                  - Are you...?
Kun je...?                  - Can you...?
```

**Real Examples:**
- Hoe is het? (How are you?)
- Wat gaan we doen? (What are we going to do?)
- Waar is de föhn? (Where is the hairdryer?)
- Wanneer kunt u langskomen? (When can you come by?)
- Heb je ervaring? (Do you have experience?)

---

#### **PATTERN 5: Talking About Time**
```
[Time] + [activity]
Om [time] ...              - At [time]...
Vanmiddag/vanavond         - This afternoon/evening
Gisteren/morgen            - Yesterday/tomorrow
Deze week/volgende week    - This week/next week
```

**Real Examples:**
- Om twee uur moet ik... (At two o'clock I have to...)
- Vanmiddag ga ik zwemmen (This afternoon I'm going swimming)
- Volgende week hebben we die wedstrijd (Next week we have that match)
- Gisteren heb ik gewassen (Yesterday I washed)

---

#### **PATTERN 6: Expressing Capability/Possibility**
```
Ik kan (niet)...           - I can (not)...
Dat kan (niet)             - That's (not) possible
Dat gaat (niet)            - That works/(doesn't) work
Dat lukt (niet)            - That succeeds/(doesn't) succeed
```

**Real Examples:**
- Ik kan niet, want... (I can't, because...)
- Dat gaat dus niet (So that doesn't work)
- Dat lukt niet (That won't work)
- Hij kan het maken (He can fix it)

## SECTION 3: CRITICAL GRAMMAR PATTERNS (25% Focus)

### PATTERN 1: Basic Sentence Word Order
```
MAIN CLAUSE: Subject + Verb + Rest
Ik     werk   in Amsterdam
I      work   in Amsterdam

INVERSION (time/place first): Time/Place + Verb + Subject + Rest
Morgen    ga     ik    naar Amsterdam
Tomorrow  go     I     to Amsterdam
```

**Practice Examples:**
- Ik woon in Nederland (I live in the Netherlands)
- Vandaag werk ik niet (Today I don't work)
- In Amsterdam woon ik (In Amsterdam I live)

---

### PATTERN 2: Subordinate Clause Word Order (EXAM-CRITICAL!)
```
Main clause, + omdat/als/dat + subject + rest + VERB
Ik kan niet,   omdat         ik       ziek    ben
I can't,       because       I        sick    am
```

**Common subordinating conjunctions:**
- omdat (because)
- als (if/when)
- dat (that)
- toen (when - past)
- voordat (before)
- nadat (after)
- terwijl (while)

**Real Examples from Dialogues:**
- Ik ga niet mee, omdat ik ziek **ben** (verb at end!)
- Als je tijd **hebt**, kunnen we afspreken
- Ik denk dat het mooi **is**
- Toen ik jong **was**, speelde ik voetbal

**MEMORIZATION TIP:** Learn these as complete phrases, not rules!

---

### PATTERN 3: Perfect Tense (Past Actions)
```
hebben/zijn + past participle
Ik heb    gewerkt     (I have worked)
Ik ben    gegaan      (I have gone)
```

**Movement verbs use ZIJN:**
- gaan → ben gegaan (have gone)
- komen → ben gekomen (have come)
- fietsen → ben gefietst (have cycled)
- lopen → ben gelopen (have walked)

**Most other verbs use HEBBEN:**
- werken → heb gewerkt (have worked)
- eten → heb gegeten (have eaten)
- maken → heb gemaakt (have made)

## SECTION 5: EXAM STRATEGY TIPS

### A2 Speaking Exam
1. **Use these high-frequency starters:**
   - "Ik denk dat..." (I think that...)
   - "Volgens mij..." (In my opinion...)
   - "Dat is lastig omdat..." (That's difficult because...)
   - "Dat lijkt me leuk/moeilijk" (That seems nice/difficult to me)

2. **Time-buying phrases when you need to think:**
   - "Nou..." (Well...)
   - "Laat me even denken..." (Let me think...)
   - "Hoe zal ik het zeggen..." (How should I say it...)
   - "Eh..." (acceptable in Dutch!)

3. **If you don't understand:**
   - "Kunt u dat herhalen?" (Can you repeat that?)
   - "Wat betekent [word]?" (What does [word] mean?)
   - "Kunt u wat langzamer praten?" (Can you speak slower?)

### A2 Listening Exam
- Focus on **W-words**: Wie (who), Wat (what), Waar (where), Wanneer (when), Waarom (why)
- Listen for time expressions: morgen, gisteren, volgende week
- Modal verbs signal important information: moet, kan, wil

### A2 Reading Exam
- Scan for names, dates, times, places first
- Look for cognates (similar to English)
- Don't panic if you don't know every word - context helps!

### A2 Writing Exam
- Use simple sentences with patterns you KNOW
- Better to write correct simple sentences than attempt complex ones
- Include time markers: eerst (first), dan (then), daarna (after that), tot slot (finally)
