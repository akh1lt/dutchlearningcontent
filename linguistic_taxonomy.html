Purpose: Classify 1,265 Dutch sentences by linguistic features to enable clustering by structural similarity and pattern extraction.

Focus: Language structure, morphology, syntax, and semantic function - NOT pedagogical strategy.

TAGGING APPROACH:
Tag each sentence with ALL applicable categories from the taxonomy above. Multiple tags per sentence are expected and encouraged for clustering analysis.

GOAL: After tagging, cluster sentences by shared linguistic features to extract ~100 high-frequency structural patterns for memorization.
