### Pattern 2: **"Kun je/jij ...?"** - Can you...? (asking if someone can do something)

## 7. GRAMMATICAL STRUCTURES OBSERVED

### Word Order in Questions:
```
Verb + subject + rest
"Ga je vanmiddag mee zwemmen?"
"Kun jij Joris vanmiddag naar het hockeyveld brengen?"
```

### Word Order with "omdat" (subordinate clause):
```
Main clause + omdat + subject + ... + VERB at END
"We gaan hardlopen, omdat we volgende week die wedstrijd hebben"
                                                              ^^ verb at end
```

### Modal verb + infinitive at end:
```
"Ik moet Joris vanmiddag naar hockeytraining brengen"
              ^^ modal           infinitive at end ^^
"We moeten trainen"
```
