1. COMMUNICATIVE INTENT (Primary Category)

What is the speaker trying to DO with this sentence?

1.1 Requesting & Asking

Learning Value: Bread-and-butter patterns for daily survival
• Direct requests: "Kun je mij helpen?" / "Mag ik...?"
• Polite requests: "Zou je...?" / "Wil je alsjeblieft...?"
• Permission seeking: "Mag ik...?" / "Is het goed als...?"
• Offers of help: "Zal ik...?" / "Kan ik iets voor je doen?"
Exam Focus: A2 speaking tasks constantly require polite requesting

1.2 Information Seeking (Questions)

Learning Value: High-frequency patterns for getting answers
• Yes/No questions: "Heb je...?" / "Is er...?" / "Kun je...?"
• Wh-questions: "Waar is...?" / "Wanneer...?" / "Hoe...?" / "Wat voor...?"
• Indirect questions: "Weet je waar...?" / "Kun je me zeggen hoe...?"
• Clarification: "Bedoel je...?" / "Wat betekent...?"
Exam Focus: Essential for KNM exam listening/speaking sections

1.3 Stating & Informing

Learning Value: Foundational declarative patterns
• Simple statements: "Ik woon in..." / "Dat is..."
• Describing situations: "Er is/zijn..." / "Het gaat om..."
• Giving information: "Het station is..." / "De winkel sluit om..."
• Reporting facts: "Ik heb gehoord dat..." / "Het is belangrijk dat..."
Exam Focus: Reading comprehension and written communication

1.4 Expressing Needs & Wants

Learning Value: Critical for service interactions
• Direct needs: "Ik heb ... nodig" / "Ik moet..."
• Preferences: "Ik wil graag..." / "Ik zou liever..."
• Wishes: "Ik hoop dat..." / "Ik zou graag willen..."
• Problems: "Ik heb een probleem met..." / "Het lukt me niet om..."
Exam Focus: Healthcare, housing, workplace scenarios

1.5 Reacting & Responding

Learning Value: Natural conversation flow
• Agreement: "Ja, dat klopt" / "Ik ben het met je eens"
• Disagreement: "Nee, dat is niet zo" / "Ik denk het niet"
• Expressing surprise: "Echt waar?" / "Dat wist ik niet"
• Confirming understanding: "Oké, ik begrijp het" / "Dus je bedoelt..."
Exam Focus: Speaking fluency and natural interaction

1.6 Suggesting & Advising

Learning Value: Social interaction patterns
• Suggestions: "Zullen we...?" / "Wat dacht je van...?" / "Misschien kunnen we..."
• Advice: "Je moet..." / "Het is beter om..." / "Je kunt het beste..."
• Recommendations: "Ik raad je aan om..." / "Je zou moeten..."
Exam Focus: Social integration scenarios

1.7 Explaining & Reasoning

Learning Value: Connecting ideas logically
• Giving reasons: "omdat..." / "want..." / "daarom..."
• Purpose: "om ... te" / "zodat..."
• Conditions: "als..." / "als je ... dan..."
• Contrast: "maar..." / "hoewel..." / "toch..."
Exam Focus: Extended responses in speaking exam

1.8 Commands & Instructions

Learning Value: Following/giving directions
• Direct imperatives: "Ga naar..." / "Neem..."
• Softened commands: "Doe ... maar" / "Kom even hier"
• Instructions: "Je moet eerst... en dan..." / "Let op dat..."
• Prohibitions: "Niet ... !" / "Je mag niet..."
Exam Focus: Following directions, workplace safety

2. VERB PATTERNS (Grammar Core)

What structural pattern does the verb create?

2.1 Single Finite Verb (Simple Present/Past)

Learning Value: Foundation patterns
• Present: "Ik werk hier" / "Hij woont daar"
• Simple past: "Ik werkte" / "Hij was" / "Ze hadden"
Pattern Recognition: Subject + finite verb + rest
Frequency: 40-50% of daily sentences

2.2 Perfect Tense (Hebben/Zijn + Participle)

Learning Value: Past experiences, completed actions
• With hebben: "Ik heb gewerkt" / "Ze heeft gebeld"
• With zijn: "Ik ben gegaan" / "Ze zijn geweest"
Pattern Recognition: Auxiliary position + participle at end
Exam Focus: Describing past events in speaking

2.3 Modal Verb Constructions

Learning Value: Expressing ability, permission, obligation
• Kunnen: "Ik kan zwemmen" / "Kun je dat doen?"
• Mogen: "Mag ik...?" / "Je mag niet..."
• Moeten: "Ik moet werken" / "Je moet dat doen"
• Willen: "Ik wil graag..." / "Wil je...?"
• Zullen: "Zullen we...?" / "Ik zal het doen"
Pattern Recognition: Modal + infinitive at end
Frequency: Top 10 most common patterns

2.4 Gaan + Infinitive (Near Future)

Learning Value: Plans and intentions
• Future plans: "Ik ga morgen..." / "We gaan ...en"
• Intentions: "Ik ga proberen om..."
Pattern Recognition: Gaan conjugated + infinitive at end
Exam Focus: Discussing plans in speaking

2.5 Te + Infinitive (Purpose/Obligation)

Learning Value: Expressing goals and requirements
• Om te: "Ik ben hier om te leren" / "Het is moeilijk om te..."
• Te + infinitive: "Ik heb geen tijd te verliezen"
• Adjective + te: "Het is belangrijk te weten"
Pattern Recognition: Watch for "om te" vs. bare "te"

2.6 Progressive (Aan het + Infinitive)

Learning Value: Ongoing actions
• "Ik ben aan het werken" / "Ze zijn aan het eten"
Pattern Recognition: Zijn + aan het + infinitive
Note: Less frequent than English progressive, but distinctive

2.7 Separable Verbs

Learning Value: Common in everyday speech
• Main clause: "Ik bel je straks op" / "Kom je mee?"
• Subordinate: "...omdat ik je opbel"
• With modal: "Ik wil je opbellen"
Pattern Recognition: Particle position changes
High-Frequency Examples: meegaan, opbellen, aankomen, uitgaan

3. WORD ORDER PATTERNS (Structural)

3.1 Standard Main Clause (V2)

Learning Value: Foundation structure
Pattern Recognition: Verb always in second position

3.2 Subordinate Clause (Verb-Final)

Learning Value: Complex sentence building
Pattern Recognition: All verbs cluster at end
Common Error Point: Learners forget verb-final order

3.3 Question Word Order

Learning Value: Information gathering
Pattern Recognition: Inversion after question word

3.4 Existential ER Constructions

Learning Value: Presenting new information
"Er is..." / "Er zijn..." / "Er komt..." / "Er staat..."
Exam Focus: Descriptions in speaking/writing

4-12. ADDITIONAL CATEGORIES

4. TIME FRAME: Present Situations, Past Events, Future Plans, Time Connectors

5. MODAL & ATTITUDE MARKERS: Certainty/Possibility, Necessity/Obligation, Politeness, Contrast/Emphasis

6. NEGATION PATTERNS: Verb negation (niet), Noun negation (geen), Never/Nothing/Nobody

7. CONNECTING & COMBINING: Coordination, Subordination, Relative Clauses

8. PRACTICAL SITUATION CLUSTERS: Healthcare, Housing, Work, Shopping, Transportation, Social/Family, Education, Money/Admin

9. PRONOUNS & REFERENCE: Personal, Possessive, Demonstratives, Reflexive/Reciprocal

10. REGISTER & TONE MARKERS: Formality Level, Directness vs. Politeness, Emotional Expression

11. SPECIAL CONSTRUCTIONS: ER forms, Pronominal Adverbs, Comparatives, Fixed Expressions

12. COMMON IRREGULAR VERBS: Zijn/Hebben/Worden, Top Modals, High-Frequency Strong Verbs

TAGGING STRATEGY

Primary Tag (Required)

Choose ONE from:

  • Communicative Intent (what is speaker doing?)
  • Verb Pattern (structural core)
  • Word Order Pattern (sentence structure)

Secondary Tags (1-3 recommended)

  • Time Frame (when?)
  • Modal/Attitude (stance?)
  • Practical Situation (real-world context)
  • Special Construction (distinctive features)

Tertiary Tags (Optional)

  • Register/Tone (if marked)
  • Pronoun Type (if teaching point)
  • Negation (if present)
  • Connection Type (if complex sentence)

Example Tagging

Sentence: "Zou je morgen even naar de dokter kunnen bellen?"

Primary: Requesting (polite request)

Secondary:

  • Modal verb construction (zullen + kunnen + infinitive)
  • Healthcare situation
  • Politeness markers (zou + even)

Tertiary:

  • Future time frame (morgen)
  • Separable verb (opbellen)

PRIORITIZATION GUIDE FOR LEARNERS

TIER 1: Survival Patterns (Weeks 1-4)

Learn these first for immediate communication:

  • Simple present declaratives
  • Yes/No questions with hebben/zijn
  • Modal verb + infinitive (kunnen, mogen, moeten, willen)
  • Basic wh-questions (waar, wat, wanneer, hoe)
  • Polite requests (mag ik, kun je)
  • Negation with niet/geen
  • Present perfect with hebben (past events)

TIER 2: Daily Fluency (Weeks 5-8)

Build on foundation:

  • Subordinate clauses with omdat/als/dat
  • Separable verbs (common ones)
  • Gaan + infinitive (future plans)
  • ER constructions (existential)
  • Time connectors (eerst, dan, daarna)
  • Coordination with en/maar/want
  • Common modal particles (maar, even, toch)

TIER 3: Exam Readiness (Weeks 9-12)

Refine and expand:

  • Te + infinitive constructions
  • Relative clauses (die/dat)
  • Complex subordination
  • Pronominal adverbs (ervoor, eraan, etc.)
  • Past tense (simple past for common verbs)
  • Advanced politeness strategies
  • Situation-specific vocabulary clusters
